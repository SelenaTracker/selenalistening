Política de Privacidade

Última atualização: 01 de Janeiro de 2024

1. Informações Coletadas

Coletamos as seguintes informações:

  • Informações de conta: Email (apenas para login)
  • Dados de uso: Músicas votadas, cálculos realizados, missões completadas
  • Dados técnicos: Tipo de navegador, dispositivo, endereço IP (para segurança)

2. Uso das Informações

Usamos suas informações para:

  • Prover funcionalidades do site (votação, ranking, calculadora)
  • Manter a segurança do site
  • Melhorar a experiência do usuário
  • Enviar notificações importantes (quando necessário)

3. Armazenamento de Dados

Seus dados são armazenados localmente no seu navegador (localStorage) e não são enviados para nossos servidores. Isso significa:

  • Seus dados ficam apenas no seu dispositivo
  • Se limpar o cache do navegador, perderá seus dados
  • Não temos acesso aos seus dados pessoais

4. Cookies e LocalStorage

Usamos localStorage do navegador para:

  • Manter sua sessão de login
  • Armazenar suas preferências
  • Salvar seu progresso no sistema de pontos
  • Manter o histórico de suas ações

Você pode limpar esses dados a qualquer momento nas configurações do seu navegador.

5. Compartilhamento de Dados

Não vendemos, alugamos ou compartilhamos seus dados pessoais com terceiros. Seus dados permanecem apenas no seu dispositivo.

6. Segurança

Implementamos medidas de segurança para proteger suas informações:

  • Dados armazenados apenas localmente
  • Senhas são armazenadas de forma segura (hash)
  • Proteção contra manipulação de dados

7. Seus Direitos

Você tem o direito de:

  • Acessar os dados que coletamos sobre você
  • Corrigir informações incorretas
  • Solicitar a exclusão de seus dados
  • Exportar seus dados em formato legível

Para exercer estes direitos, entre em contato conosco.

8. Dados de Menores

Nosso site não coleta intencionalmente dados de menores de 13 anos. Se descobrirmos que coletamos dados de uma criança sem consentimento parental, removeremos essas informações imediatamente.

9. Alterações na Política

Podemos atualizar esta política periodicamente. A versão mais recente estará sempre disponível nesta página.

10. Contato

Se tiver dúvidas sobre esta política de privacidade, entre em contato através da nossa página de contato.
